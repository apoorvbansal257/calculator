# Calculator

### This project is an advanced calculator made using C++ language. Following operations are included in this project:
* Arithmetic
  * Addition 
  * Subtraction
  * Multiplication
  * Division
* Trignometric
  * Sine 
  * Cosine
  * Tangent
  * Cosecant
  * Secant
  * Cotangent
* Logarithmic
  * Base e
  * Base 10
* Exponential
  * Base e
  * Custom Base
